# JavaScript 知识点总结(一)

- 对象类型
除了原始类型以外都是对象类型。不同之处是原始类型存储的是值，对象类型存储的是地址。
例子：

```js
function test(person) {
  person.age = 26
  person = {
    name: 'yyy',
    age: 30
  }

  return person
}
const p1 = {
  name: 'yck',
  age: 25
}
const p2 = test(p1)
```
